    梯度下降

    July 7, 2026
    Ml, Dl

    梯度下降#

    Gradient Descent（梯度下降）是一类通过「沿目标函数负梯度方向迭代更新参数」寻找最小值的优化算法，是几乎所有机器学习与深度学习模型训练的核心机制。

    出现动机#

    很多模型的损失函数没有闭式解（神经网络），或闭式解代价太高。梯度下降提供通用做法：先随机初始化参数，反复沿「让损失下降最快」的方向迈一小步，期望逐步逼近局部最小值。

    优点缺点#

    优点：①通用、简单，几乎适用所有可微损失函数；②配合反向传播可扩展到超大规模神经网络；③有大量变体（Momentum、Adam、RMSProp）解决收敛与稳定性问题。缺点：①对学习率敏感，太大不收敛、太小收敛慢；②只能保证收敛到局部最优 / 鞍点；③存在梯度消失 / 爆炸等病态情况。

    学习率

    July 7, 2026
    Dl

    学习率#

    Learning Rate（学习率）是优化算法的核心超参数：决定每次参数更新「迈多大一步」。太大容易震荡 / 越过最优解，太小收敛过慢甚至停滞。

    出现动机#

    梯度下降沿负梯度方向更新参数：w ← w - η · ∇L。η（学习率）直接决定更新幅度。学习率是训练中「最需要调节」的超参数，几乎所有训练任务都要先调它。

    优点缺点#

    优点：①调节简单，几乎所有框架都支持按 step 调度；②配合学习率调度（warmup + decay）效果显著；③是模型训练中最常调节的超参数之一。缺点：①对训练稳定性影响极大，调不好就崩；②不同任务 / 模型需要不同值，没有通用最佳；③对极大 / 极小 batch size 都需重新调节。

    数据增强#

    Data Augmentation（数据增强）通过对现有数据做轻微变换（旋转、裁剪、翻转、加噪、Mixup 等）人工生成新的训练样本，是扩充数据集、提升模型泛化能力的常用技巧。

    出现动机#

    深度学习需要大量数据，但标注成本高。数据增强用「对原始数据做合理扰动」免费生成新样本：图像可以水平翻转 / 旋转 / 裁剪 / 调色，文本可以同义词替换 / 回译 / 随机删除，音频可以加噪 / 变速。

    优点缺点#

    优点：①几乎零成本扩大训练集；②显著提升模型对平移、旋转、噪声的鲁棒性；③与半监督、自监督学习结合效果更好。缺点：①不是所有变换都「合理」：医学影像不能水平翻转，文本不能颠倒语序；②过度的增强可能损害语义；③不同任务需要专门设计增强策略。

    人工神经网络

    July 7, 2026
    Dl

    人工神经网络#

    Artificial Neural Network（人工神经网络，ANN）是一类模仿生物神经元连接方式的计算模型：由大量「神经元」节点和带「权重」的连接构成，按层组织（输入 / 隐藏 / 输出），通过训练调整权重让模型逼近目标函数。

    出现动机#

    1943 年 McCulloch 和 Pitts 提出神经元数学模型，1986 年 Rumelhart 等人把反向传播用于多层网络——奠定了 ANN 的理论基础。它的核心思想是「大量简单单元的连接可以涌现出复杂智能」，是深度学习的载体。

    优点缺点#

    优点：①通用逼近能力，能拟合任意复杂函数；②可堆叠到任意深度，能力随规模平滑提升；③统一框架适配图像 / 语音 / 文本 / 多模态。缺点：①训练需要大量数据；②对超参敏感；③可解释性弱。

    术语别名#

    感知机

    July 7, 2026
    Dl

    感知机#

    Perceptron（感知机）是 Frank Rosenblatt 在 1957 年提出的最简单人工神经元模型：接收多个输入，按权重加权求和后通过阶跃激活函数输出 0 或 1。它是神经网络的基本构件。

    出现动机#

    感知机是神经网络的「最小单元」：把所有计算简化成一个线性加权求和 + 一个非线性阈值激活。它能解决线性可分问题（如 AND、OR），但无法解决 XOR——这一点直接启发了多层感知机（MLP）的诞生。

    优点缺点#

    优点：①结构简单、可解释，是理解神经网络最直观的入口；②训练算法（Perceptron Learning Rule）直观可推导；③奠定了现代神经元的「线性 + 激活」基本范式。缺点：①只能解决线性可分问题；②阶跃激活函数不可微，无法用梯度下降；③已被多层感知机（MLP）取代。

    权重

    July 7, 2026
    Dl

    权重#

    Weights（权重）是神经网络中两个神经元之间连接的「强度」，是模型通过训练学习的核心参数。模型学到的所有「知识」都编码在权重矩阵中。

    出现动机#

    神经元之间不是均等连接的——有的连接重要、有的不重要。权重把这个「重要性」量化成一个数值，让模型能区分不同输入特征的影响。

    优点缺点#

    优点：①是模型学习能力的载体；②训练就是调整权重的过程；③权重分析（可视化、剪枝、量化）有助于理解模型。缺点：①大模型权重数量动辄百亿，存储与推理成本高；②随机初始化对训练稳定性影响大；③对抗样本可通过微小扰动让权重组合失效。

    偏置

    July 7, 2026
    Dl

    偏置#

    Biases（偏置）是神经元中加在加权和上的「额外常数项」，用来平移激活函数的输出，使神经元能更好地拟合数据。它和权重一起构成神经网络的可学习参数。

    出现动机#

    线性模型 y = wx 没有偏置时只能过原点。给每个神经元加一个 bias，相当于给它一个「基准激活水平」，让模型对常数项 / 偏移量也能拟合——是几乎所有神经网络的标配。

    优点缺点#

    优点：①让模型能拟合不通过原点的函数；②是模型表达力的关键之一；③实现简单（每个神经元一个标量）。缺点：①与权重一起需要被优化，增加参数；②对偏置的正则化通常不需要；③不影响神经元的「激活模式」。

    ReLU

    July 7, 2026
    Dl

    ReLU#

    ReLU（Rectified Linear Unit，修正线性单元）是最常用的激活函数：f(x) = max(0, x)。正区间导数为 1、计算极简、缓解梯度消失，是深度学习「复兴」的关键组件。

    出现动机#

    Sigmoid / Tanh 在饱和区梯度接近 0，导致深层网络难以训练。ReLU 在正区间保持梯度恒为 1，从根本上缓解了梯度消失；同时计算只需一个 max 操作，速度极快。

    优点缺点#

    优点：①计算简单（一次 max）；②正区间梯度恒为 1，深层网络可训练；③实验效果稳健，是 CNN、MLP 等网络的默认选择。缺点：①负区间输出恒为 0，神经元可能「死亡」且不可逆；②输出均值不为 0，可能影响收敛；③对输入分布敏感，需要配合 BatchNorm。

    术语别名#

    • ReLU
    • Rectified Linear Unit
    • 修正线性单元
